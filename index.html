APP INFO

This application was made using the following tools:

  • HTML
  • CSS
  • Bootstrap
  • Javascript

The main function of this application is to log the time stamp of the employees.

Only the registered employees can use the application. The admin can add and update the record if needed.

However, the app has no delete function for record keeping purposes.

Lunch Policy

The employee must clock-out to take his/her schedule lunch and clock-in to return to work.

For Future Improvements

The app is a segment of a payroll system and recommended to add more functions

such computing the total hours of the employee,

creating a timeframe for regular pay and if exceeds will considered overtime rate,

and many more.
